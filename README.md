## **🚀 Getting Started**  
### **Prerequisites**  
- Node.js `v18+`

1. **Clone the repo**  
   ```bash
   git clone https://github.com/SangerForCode/Health_App.git
   cd Health_App
   ```
2. **Install dependencies**  
   ```bash
   npm install
   ```
3. **Run the app**
